This project delivers an automated pipeline and executive dashboard for analyzing ChatGPT user sentiment.

Key Technical Components:

Machine Learning: Uses a Logistic Regression model with TF-IDF vectorization to classify reviews as Positive, Neutral, or Negative.

Data Processing: Includes a Python-based cleaning script to standardize text and prepare raw datasets for analysis.

Business Intelligence: An interactive Streamlit dashboard that provides real-time KPIs (Total Reviews, Average Rating) and performance tracking.

Management Value:

Granular Insights: Filter data by platform (App Store, Web, etc.) and software version to pinpoint specific issues.

Voice of Customer: Visualizes key user pain points and praises through automated word clouds.

Data-Driven Decisions: Enables leadership to prioritize product updates based on quantifiable user feedback trends.
